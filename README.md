# 🧮 BMI Calculator with Weight Guide

![CSS](https://img.shields.io/badge/CSS-3-blue?style=for-the-badge&logo=css3)

A simple and interactive **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**.  
This project showcases strong fundamentals of **DOM Manipulation**, event handling, and dynamic UI updates.

---

## 🚀 Features

✔️ Calculate BMI using height and weight  
✔️ Instant result display  
✔️ Weight category message (Underweight, Normal, Overweight, Obese)  
✔️ Input validation  
✔️ Responsive UI  

---

## 📊 BMI Categories

| BMI Range   | Category      |
|-------------|---------------|
| < 18.5      | Underweight   |
| 18.5 – 24.9 | Normal weight |
| 25 – 29.9   | Overweight    |
| 30+         | Obese         |

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (DOM Manipulation)
